position in css!!

  1. static: The originial or default position of an element
  2. relative: The element is positioned relative to its normal or original position same as static position, here just we can move our elements by adding offset values like top,bottom,right,left. Element stays in normal document flow.
  3. Element is positioned relative to its parent.. when parent not present element is positioned relative to main document.. Element is removed from the normal document flow.. means no space is created for that element
  4. fixed: Element is positioned relative to the viewport.. Element doesnt move and stays in same position even after scrolling also.. Element is removed from the normal flow.
  5. position always comes after the margin inside box model sticky:Element is in normal position till it reaches to given value lets say top=0px after that it remains sticky or fixed event after scrolling also
